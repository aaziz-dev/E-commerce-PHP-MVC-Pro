# Système E-commerce Pro (Architecture MVC)

## Présentation
Ce projet est une application de gestion e-commerce complète développée en **PHP natif** avec une approche **Orientée Objet (POO)** et une architecture **MVC**.

## Stack Technique
- **Langage :** PHP 8.x
- **Architecture :** MVC (Modèle-Vue-Contrôleur)
- **Base de données :** MySQL (PDO pour la sécurité)
- **Frontend :** HTML5 / CSS3 / JavaScript

## 📁 Structure du Projet
- `/classes` : Logique métier et entités (Utilisateur, Produit, Commande).
- `/controllers` : Gestion de la logique applicative.
- `/models` : Interactions avec la base de données.
- `/views` : Interface utilisateur.
- `/database` : Script SQL pour la création de la base de données.

## ⚙️ Installation
1. Cloner le dépôt.
2. Importer le fichier `./database/unicap.sql` dans votre serveur MySQL.
3. Configurer les accès à la base de données dans le fichier de configuration.
